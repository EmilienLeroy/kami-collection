## Usage

Install the component using ``npm`` or ``yarn`` or ``pnpm`` :

```console
$ npm install --save @kamiapp/pdf
```

Import the component into your **main** js/ts file :

```js
import '@kamiapp/pdf';
```
And use the component directly into your html :

```html
<kami-pdf></kami-pdf>
```
